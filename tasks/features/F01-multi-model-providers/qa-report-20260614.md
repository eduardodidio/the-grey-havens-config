## 4. Open items — investigated and resolved

### 4.1 Non-F01 test regressions (F10 / F12 / F14)

Investigated by diffing against a pre-F01 baseline (`git stash` of the
F01-touched tracked template files):

- **Root cause**: F01's `compile-skills` regeneration of
  `templates/.claude/commands/*.md` and `templates/agents/prompts/*.md`
  added a `<!-- GENERATED FILE — DO NOT EDIT ... -->` header and
  resynced content from `skills/`. This **fixed** a large amount of
  *pre-existing* drift between `templates/.claude/commands/` and
  `templates/commands/` (the actual symlink target of the live
  `.claude/commands`): F14 went from 27 failures (pre-F01 baseline) down
  to 4. However, the header itself was not propagated to
  `templates/commands/*.md`, leaving 6 files with a header-only diff —
  causing the remaining F10 (3) and F14 (4) failures.
- **Fix applied**: added the same `GENERATED FILE` header block to
  `templates/commands/{check-readiness,create-feature,didio,brainstorm,
  research,product-brief}.md` so they match
  `templates/.claude/commands/*.md` exactly.
  - F10-ac7-bypass-test.sh: now **PASS** (9/9, was 3 fails)
  - F14-commands-smoke.sh: now **PASS** (50/50, was 4 fails)
- **F12-sync-archive-smoke.sh**: scenario 2 expects `NO_CHANGE` for
  `didio.config.json` when the target already has "all template
  top-level keys". F01's schema change (T01/T03) added two new
  top-level keys to `templates/didio.config.json`: `providers` and
  `_comment_provider_per_role`. The test fixture
  (`tests/F12-sync-archive-smoke.sh`) didn't include them, so sync
  detected a diff and the `NO_CHANGE` assertion failed.
  - **Fix applied**: added the two new keys (matching
    `templates/didio.config.json`) to the fixture config in
    `tests/F12-sync-archive-smoke.sh`.
  - F12-sync-archive-smoke.sh: now **PASS** (9/9, was 1 fail)

All three regressions were caused by F01's skill-compile/config-schema
changes as suspected, and are now fixed with surgical, scoped edits
(no behavioral code changes — only template/fixture sync).
