  `bash tests/run.sh`.
- Scope: validates the real `didio.config.json` (and `models_economy`) ‚Äî every
  role has non-empty model+fallback, every provider is registered, types correct.

  `max_parallel:"x"`) must fail clearly; inputs read via `sys.argv`/file open.

## Test scenarios

- **Happy path:** real `didio.config.json` ‚Üí valid.
